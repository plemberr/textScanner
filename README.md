Описание проекта

Данный проект представляет собой программную систему для автоматического распознавания, анализа и классификации входящих документов различных форматов.
Система предназначена для обработки неструктурированных документов (сканы, PDF, DOCX) с целью извлечения текста, выделения ключевых реквизитов и определения типа документа с использованием методов OCR, NLP и машинного обучения.

Система разделена на модули:

1. Text Extractor - Извлечение текста из PDF, изображений и DOCX (OCR и прямой парсинг)
2. NLP Extractor -	Анализ текста и извлечение реквизитов документа
3. Classifier - Классификация документов с использованием ML-модели
4. Processor - Оркестрация всех этапов обработки документа
5. Bootstrap	Инициализация и настройка внешних зависимостей
